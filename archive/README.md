# Archive Legacy Snapshot

Ce dossier est réservé aux artefacts historiques (snapshots, historiques de migration, exports).

## Règles

- Il **ne fait pas partie du périmètre actif** du produit.
- Les fichiers contenus ici sont en lecture seule par défaut.
- Les noms et contenus legacy (`socialflow`, `SocialFlow`, etc.) y sont conservés comme preuve historique.
- Pour toute migration de marque, l’archive doit être traitée en lot séparé si nécessaire.
