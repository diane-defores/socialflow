Questions fréquentes sur les tarifs

Pourquoi c'est gratuit ?

SocialFlow est un projet open-source. Le code est public, les contributions sont bienvenues. Nous croyons que la gestion de vos réseaux sociaux ne devrait pas être un service payant mensuel.

Y aura-t-il une version payante ?

Nous envisageons des fonctionnalités premium pour les équipes (partage de profils, analytics avancés). Mais les fonctionnalités actuelles resteront toujours gratuites.

Mes données sont-elles vendues ?

Non. SocialFlow ne collecte aucune donnée. Vos cookies et sessions restent sur votre appareil. Le code est open-source, vous pouvez vérifier par vous-même.
